Mathematic equation

Uses MathJax for rendering and TeX syntax for markup. Automatically detects lines enclosed with $$ and adds the library if it is detected. Only display format is supported, not single $ inline format. Note also that the equation must have newlines preceding and following it to render properly.

\[ e^{i\pi} = -1 \]

\[\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}\]
